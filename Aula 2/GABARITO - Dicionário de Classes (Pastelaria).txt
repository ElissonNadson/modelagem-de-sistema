================================================================================
                                CLASSE 1 - PRODUTO
================================================================================

Nome da Classe: Produto

Descrição: Classe que representa os itens vendidos pela pastelaria, incluindo 
pastéis, bebidas e sobremesas. Armazena informações básicas de cada produto.

┌─────────────────────────────────────────────────────────────────────────────┐
│                              ATRIBUTOS                                       │
├──────────────────┬────────────────┬───────────────┬─────────────────────────┤
│      Nome        │      Tipo      │ Visibilidade  │       Descrição         │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ codigo           │ int            │ -             │ Código único do produto │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ nome             │ String         │ -             │ Nome do produto         │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ descricao        │ String         │ -             │ Descrição do produto    │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ precoUnitario    │ double         │ -             │ Preço de venda          │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ categoria        │ String         │ -             │ Tipo: pastel/bebida/etc │
└──────────────────┴────────────────┴───────────────┴─────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                               MÉTODOS                                        │
├──────────────────────────┬────────────────┬───────────────┬─────────────────┤
│          Nome            │    Retorno     │ Visibilidade  │   Descrição     │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ getCodigo()              │ int            │ +             │ Retorna código  │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ getNome()                │ String         │ +             │ Retorna nome    │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ getPrecoUnitario()       │ double         │ +             │ Retorna preço   │
└──────────────────────────┴────────────────┴───────────────┴─────────────────┘

Relacionamentos: 
- Associação com ItemPedido (0..* - um produto pode estar em vários itens)
- Superclasse de Pastel (generalização)

│                  │                │               │                         │
└──────────────────┴────────────────┴───────────────┴─────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                               MÉTODOS                                        │
├──────────────────────────┬────────────────┬───────────────┬─────────────────┤

├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ getSabor()               │ String         │ +             │ Retorna sabor   │

================================================================================
                                CLASSE 4 - PEDIDO
================================================================================

Nome da Classe: Pedido

Descrição: Classe que registra os pedidos realizados pelos clientes, contendo
informações sobre data, status e valor total.

┌─────────────────────────────────────────────────────────────────────────────┐
│                              ATRIBUTOS                                       │
├──────────────────┬────────────────┬───────────────┬─────────────────────────┤
│      Nome        │      Tipo      │ Visibilidade  │       Descrição         │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ numeroPedido     │ int            │ -             │ Número único do pedido  │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ dataHora         │ Date           │ -             │ Data e hora do pedido   │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ status           │ String         │ -             │ Aguardando/Preparo/etc  │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ tipo             │ String         │ -             │ Balcão ou Delivery      │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ valorTotal       │ double         │ -             │ Valor total do pedido   │
└──────────────────┴────────────────┴───────────────┴─────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                               MÉTODOS                                        │
├──────────────────────────┬────────────────┬───────────────┬─────────────────┤
│          Nome            │    Retorno     │ Visibilidade  │   Descrição     │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ calcularTotal()          │ double         │ +             │ Calcula total   │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ atualizarStatus()        │ void           │ +             │ Muda status     │
└──────────────────────────┴────────────────┴───────────────┴─────────────────┘

Relacionamentos: 
- Associação com Cliente (1 pedido pertence a 1 cliente)
- Associação com Funcionario (1 pedido é atendido por 1 funcionário)
- Composição com ItemPedido (1 pedido tem 1..* itens)
- Superclasse de PedidoDelivery (generalização)

================================================================================
                        CLASSE 6 - ITEM PEDIDO (ASSOCIATIVA)
================================================================================

Nome da Classe: ItemPedido (Classe Associativa)

Descrição: Classe associativa que representa cada produto dentro de um pedido,
armazenando a quantidade solicitada. Surge da associação entre Pedido e Produto.

┌─────────────────────────────────────────────────────────────────────────────┐
│                              ATRIBUTOS                                       │
├──────────────────┬────────────────┬───────────────┬─────────────────────────┤
│      Nome        │      Tipo      │ Visibilidade  │       Descrição         │
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ quantidade       │ int            │ -             │ Qtd do produto no pedido│
├──────────────────┼────────────────┼───────────────┼─────────────────────────┤
│ subtotal         │ double         │ -             │ Preço x quantidade      │
└──────────────────┴────────────────┴───────────────┴─────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                               MÉTODOS                                        │
├──────────────────────────┬────────────────┬───────────────┬─────────────────┤
│          Nome            │    Retorno     │ Visibilidade  │   Descrição     │
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ calcularSubtotal()       │ double         │ +             │ Calcula subtotal│
├──────────────────────────┼────────────────┼───────────────┼─────────────────┤
│ getQuantidade()          │ int            │ +             │ Retorna qtd     │
└──────────────────────────┴────────────────┴───────────────┴─────────────────┘

Relacionamentos: 
- Associação com Pedido (1..* itens pertencem a 1 pedido)
- Associação com Produto (cada item referencia 1 produto)
- É uma CLASSE ASSOCIATIVA entre Pedido e Produto

================================================================================
                          LEGENDA DE VISIBILIDADE
================================================================================

    +  (público)   = Qualquer classe pode acessar
    -  (privado)   = Apenas a própria classe acessa
    #  (protegido) = Classe e subclasses acessam

================================================================================
                            TIPOS COMUNS
================================================================================

    String   = Texto
    int      = Número inteiro
    double   = Número decimal
    boolean  = Verdadeiro/Falso
    Date     = Data
    void     = Sem retorno (para métodos)
